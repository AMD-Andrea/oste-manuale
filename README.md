# Guida OSTE — Gestionale per la Ristorazione

Benvenuto nella guida ufficiale di **OSTE**, il gestionale pensato per ristoranti e locali.

Questa guida è pensata per chi usa OSTE per la prima volta. Troverai spiegazioni semplici e chiare, senza termini tecnici complicati.

| 📋 Ordini | Come prendere e inviare le comande |
| 💳 Cassa | Come chiudere i conti e incassare |
| 📦 Magazzino | Come controllare scorte e food cost |
| 📊 Statistiche | Come leggere i dati del locale |
| 🔌 Moduli | I servizi aggiuntivi disponibili |
| ❓ FAQ | Le domande più frequenti |
| 📖 Glossario | I termini spiegati in modo semplice |
