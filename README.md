# Benvenuto

Benvenuto nella guida ufficiale di **OSTE**, il gestionale pensato per ristoranti e locali.

Questa guida è il punto di partenza ideale per chi si avvicina a OSTE per la prima volta, ma anche per chi desidera riprendere rapidamente confidenza con le sue funzionalità.\
Abbiamo pensato a contenuti chiari, diretti e accessibili a tutti, eliminando tecnicismi inutili per offrirti un’esperienza semplice, veloce ed efficace.
